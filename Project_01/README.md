# Nexus Studio - Digital Agency Website

A modern, dark-themed static website for a digital agency, built as **Project 1** for DecodeLabs Frontend Development Industrial Training.

![Nexus Studio Preview](https://via.placeholder.com/800x400/0a0a0a/00d4ff?text=Nexus+Studio+Preview)

## 🎯 Project Objective

To create a clean, professional, and fully responsive static webpage using **HTML5** and **CSS3** following industry best practices.

## ✨ Features

- Fully Responsive Design (Mobile + Desktop)
- Modern Dark Theme with vibrant accents
- Smooth hover animations and transitions
- Semantic HTML5 structure
- CSS Grid + Flexbox layout
- Professional Hero Section
- Services Showcase
- Featured Projects Gallery
- About Us Section
- Split Contact Section with form
- Fixed Navigation Bar

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling, Grid, Flexbox, Animations
- Google Fonts (Inter)
- Font Awesome Icons

## 📋 Project Requirements Fulfilled

- ✅ Semantic HTML tags (`header`, `main`, `section`, `footer`)
- ✅ Proper heading hierarchy (Single `<h1>`)
- ✅ External CSS file only
- ✅ Responsive design using CSS Grid & Flexbox
- ✅ Clean, maintainable, and well-commented code
- ✅ Professional UI/UX
