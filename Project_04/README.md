📋 Overview

DevHire PK is a multi-section tech job portal built for the Pakistani market. It features a complete job application form with validation, job listings, company profiles, and salary guides — all built with vanilla HTML, CSS, and JavaScript.

✨ Features

- **4-Step Application Form** — Personal Info → Skills → Details → Review
- **Real-Time Validation** — Email, phone (03XX-XXXXXXX), CNIC, URLs
- **Skills Tags** — Add/remove with suggested chips
- **Salary Range Slider** — Dual-handle PKR range selector
- **File Upload** — Resume upload with drag-drop (PDF/DOCX)
- **Multi-Section Navigation** — Jobs, Companies, Salaries, Apply tabs
- **Job Listings** — 6 cards with salary and quick apply
- **Company Profiles** — 6 Pakistani IT companies
- **Salary Guide** — 6 roles with bar chart visualization
- **Success Modal** — Confetti animation on submission
- **Toast Notifications** — User feedback system
- **Fully Responsive** — Mobile hamburger menu

🛠️ Tech Stack

- HTML5
- CSS3 (Variables, Grid, Flexbox, Animations)
- JavaScript (DOM, Events, Validation, State)
- Font Awesome 6

Zero dependencies. No frameworks.
