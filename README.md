ShopVault - Premium Online Marketplace
ShopVault is a fully functional e-commerce marketplace website built as part of DecodeLabs Frontend Development - Project 2. It showcases advanced CSS3 techniques including Grid, Flexbox, custom properties, and responsive design patterns. The platform features 16 products across 8 categories with dynamic filtering, sorting, shopping cart management, and wishlist functionality — all powered by vanilla JavaScript with zero dependencies.

TaskFlow - Interactive Task Manager
TaskFlow is an interactive task management application developed as part of DecodeLabs Frontend Development - Project 3. It demonstrates core JavaScript concepts including the IPO Model (Input → Process → Output), DOM manipulation, event handling, state management, and localStorage persistence. The application features CRUD operations, priority levels, category filtering, dark mode theming, real-time statistics, and keyboard shortcuts — all built with clean separation of HTML, CSS, and JavaScript.

DevHire PK - Tech Job Application Portal
DevHire PK is a complete multi-section job application platform built as part of DecodeLabs Frontend Development - Project 4. It showcases advanced form design and validation techniques including real-time input validation, multi-step form processing, file upload with drag-and-drop, and custom validators for Pakistani formats (CNIC, phone numbers). The platform features job listings, company profiles, salary guides, an interactive skills selector, dual-range salary slider, and a full application review system — demonstrating mastery of form handling, DOM manipulation, and user experience design with vanilla JavaScript.
